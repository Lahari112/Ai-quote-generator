# AI Quote Generator
 
A simple React app that generates random inspirational quotes using a public API.

## Live Demo

https://ai-quote-generator-nine.vercel.app/

## Features

- Generate random quotes
- Fetch data from API
- Responsive centered UI
- Built with React + Vite
- Simple and clean design

## Tech Stack

- React
- JavaScript
- Vite
- CSS

## What I Learned

- React basics
- useState
- useEffect
- API fetching
- async/await
- Component rendering
- Deploying with Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ai-quote-generator.git
```

Go inside project folder:

```bash
cd ai-quote-generator
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

## Deployment

Deployed using Vercel.
